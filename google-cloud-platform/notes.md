### Google App Engine
* PaaS - Platform as a Service
* Similar to AWS Elastic Beanstalk
* Fully managed serverless application platform managed by Google
* Easy to configure
	* versioning
	* security
	* scaling
* Can do following with `stackdriver`
	* Monitoring
	* Logging
	* Diagnostics
